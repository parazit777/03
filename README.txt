LLM Data Analyst

Простое веб-приложение для анализа CSV-файлов с помощью ИИ (LLM).

Особенности

- Удобный интерфейс на Streamlit
- Поддержка бесплатных моделей OpenRouter
- Два режима: полный автоматический анализ + ответы на вопросы
- Полностью на русском языке

Как запустить

1. Клонируйте репозиторий(03)
- файл "app.py"
2. Запустите приложение

-в терминале "streamlit run app.py"

3. Откроется web интерфейс. Загрузите свой .csv файл, с которым хотите работать.
4. Получите бесплатный API-ключ на https://openrouter.ai/keys и вставьте его в поле при запуске приложения.
5. Выберите модель LLM и запустите анализ приложения.(в случае каких либо ошибок, попробуйте перезапустить анализ несколько раз)
